# RUNWITHBOCA Website

A modern, sleek website for a South Florida running club based in Boca Raton, Florida.

## Project Overview

RUNWITHBOCA is a community-focused running club website that showcases group runs, training sessions, and local running routes along the beautiful Boca Raton coastline. The site features a clean, minimalist design with a sunset beach aesthetic.

## Features

### Design Elements
- **Modern Hero Section**: Beautiful sunset beach imagery that captures the essence of coastal Florida running
- **Responsive Layout**: Fully responsive design that works seamlessly across desktop, tablet, and mobile devices
- **Custom Favicon**: Running-themed icon that appears in browser tabs
- **Smooth Scrolling**: Enhanced navigation experience with smooth scrolling between sections

### Content Sections
1. **Hero Banner**: Eye-catching sunset imagery with club branding
2. **About Section**: Information about the running club community
3. **Events Section**: Details about group runs, training sessions, and community events
4. **Resources**:
   - Interactive Google Map showing A1A Ocean Drive running routes
   - Link to detailed route guide (Palmetto to Delray Beach)
   - Contact information

### Interactive Features
- **Square Button Design**: Modern, clickable square buttons for navigation
- **Embedded Map**: Google Maps integration showing popular running routes
- **External Links**: Direct connections to Instagram, Strava, and route guides
- **Scroll-to-Section Navigation**: Smooth internal page navigation

## Design Philosophy

- **Color Scheme**: Dark backgrounds with vibrant pink/coral accents and neon green highlights
- **Typography**: Custom fonts including 'Super Vanilla', 'Orbitron', and 'Rajdhani' for a modern athletic aesthetic
- **Layout**: Clean, card-based design with generous whitespace
- **Imagery**: High-quality beach and running photos from Unsplash

## Featured Running Routes

The website highlights Boca Raton's premier running locations:

### A1A Ocean Drive Path
- **North Route**: Spanish River Park (2.5 miles), Spanish River Boulevard (3.0 miles), Sea Gate (4.5 miles), Linton Boulevard (6.0 miles), and Anchor Park (7.5 miles)
- **South Route**: Camino Real (1 mile), Mayfair Hotel (1.5 miles), Deerfield Beach Pier (2.5 miles), and Island Point (3.0 miles)

### Beach Running
- Approximately 8 miles of continuous beach from Palmetto Park Road north to Delray Beach
- Best at low tide for firmer sand and optimal running conditions
